# Data and code for [Viewpoint and Context] experiments

For each experiment the folder structure is as follows

```
Experiment
│   │   00_preprocessing.R
│   │   01_model-selection.R
│   │   02_vis-and-posthoc.R
│   │   Analysis.Rproj
│   └───data
│   │   └───processed
│   │   └───raw
│   └───models
│   └───plots
│   └───src   
```

Double click the .Rproj file to open the project in R.
